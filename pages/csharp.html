O que é C#?

C# (C Sharp) é uma linguagem de programação moderna, orientada a objetos, fortemente tipada e de alto nível, desenvolvida pela Microsoft como parte da plataforma .NET. Foi lançada no ano 2000 e criada por Anders Hejlsberg.

C# é conhecida por sua robustez, segurança de tipos e capacidade de desenvolver aplicações empresariais escaláveis. Ela combina a produtividade de linguagens de alto nível com o poder de controle de linguagens de baixo nível.

Características principais do C#:

  • Orientação a objetos
  • Fortemente tipada
  • Compilada
  • Gerenciamento automático de memória (garbage collection)
  • Integração com a plataforma .NET
  • Suporte a programação assíncrona

C# é amplamente utilizada no desenvolvimento de aplicativos Windows, aplicações web com ASP.NET, jogos com Unity, aplicativos móveis com Xamarin, e muito mais. Sua versatilidade e suporte corporativo da Microsoft a tornam uma escolha excelente para diversos tipos de projetos.

Exemplos de Código C#

Olá Mundo

// Meu primeiro programa em C#
using System;

class Program
{
    static void Main()
    {
        Console.WriteLine("Olá, Mundo!");
    }
}

Este é o exemplo mais básico de um programa C#. O método Console.WriteLine() exibe o texto "Olá, Mundo!" no console.

Variáveis e Tipos de Dados

// Variáveis em C#
using System;

class Program
{
    static void Main()
    {
        string nome = "Maria";  // string
        int idade = 25;         // inteiro
        double altura = 1.65;   // double
        bool estudante = true;  // boolean

        Console.WriteLine($"Nome: {nome}, Idade: {idade}, Altura: {altura}m, Estudante: {estudante}");
    }
}

C# é uma linguagem fortemente tipada, o que significa que você precisa declarar o tipo de cada variável. Isso ajuda a prevenir erros e melhora a performance.

Estruturas de Controle

// Estrutura condicional (if-else)
using System;

class Program
{
    static void Main()
    {
        int idade = 18;

        if (idade >= 18)
        {
            Console.WriteLine("Você é maior de idade.");
        }
        else
        {
            Console.WriteLine("Você é menor de idade.");
        }

        // Loop for
        Console.WriteLine("Contando de 1 a 5:");
        for (int i = 1; i <= 5; i++)
        {
            Console.WriteLine(i);
        }
    }
}

C# oferece estruturas de controle como if-else para tomada de decisão e loops para repetição de código, semelhantes a outras linguagens C-style.

Métodos

// Definindo métodos
using System;

class Program
{
    static void Main()
    {
        string mensagem = Saudacao("Carlos");
        Console.WriteLine(mensagem);

        Console.WriteLine(Soma(5, 3));  // 8
        Console.WriteLine(Soma(5));     // 5
    }

    static string Saudacao(string nome)
    {
        return $"Olá, {nome}! Bem-vindo ao C#!";
    }

    // Método com parâmetros opcionais
    static int Soma(int a, int b = 0)
    {
        return a + b;
    }
}

Em C#, métodos são blocos de código reutilizáveis que realizam tarefas específicas. Você pode definir métodos com parâmetros e valores padrão.
