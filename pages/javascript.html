Começar a Aprender

O que é JavaScript?

JavaScript é uma linguagem de programação de alto nível, dinâmica, interpretada e orientada a objetos, desenvolvida inicialmente pela Netscape em 1995. Foi criada por Brendan Eich e originalmente chamada de LiveScript.

JavaScript é conhecida como a linguagem de programação da web, sendo uma das tecnologias fundamentais da World Wide Web, junto com HTML e CSS. Ela permite adicionar interatividade às páginas web e executar código no navegador do usuário.

Características principais do JavaScript:

  • Linguagem interpretada (não precisa ser compilada)
  • Dinâmica e fracamente tipada
  • Suporte a programação orientada a objetos, funcional e imperativa
  • Execução no lado do cliente (navegador) e servidor (Node.js)
  • Assíncrona e baseada em eventos
  • Amplo ecossistema de bibliotecas e frameworks

JavaScript é amplamente utilizada no desenvolvimento de aplicações web interativas, single-page applications (SPAs), aplicações móveis híbridas, desenvolvimento back-end com Node.js, jogos para navegador, Progressive Web Apps (PWAs), e muito mais. Sua versatilidade e onipresença nos navegadores a tornam uma linguagem essencial para qualquer desenvolvedor web.

Exemplos de Código JavaScript

Olá Mundo

// Meu primeiro programa em JavaScript
console.log("Olá, Mundo!");

// Alternativa usando DOM para exibir no navegador
document.addEventListener("DOMContentLoaded", function() {
    document.body.innerHTML += "
Olá, Mundo!
";
});

Este é o exemplo mais básico de código JavaScript. O método console.log() exibe o texto "Olá, Mundo!" no console do navegador, enquanto a segunda parte mostra como adicionar conteúdo à página web.

Variáveis e Tipos de Dados

// Variáveis em JavaScript
// Usando let e const (ES6+)
let nome = "Maria";    // string
let idade = 25;       // number
let altura = 1.65;    // number (não há tipo separado para decimais)
let estudante = true; // boolean
const PI = 3.14159;   // constante

// Tipos de dados especiais
let nulo = null;      // valor nulo explícito
let indefinido;       // undefined
let obj = {           // objeto
    nome: "João",
    idade: 30
};
let numeros = [1, 2, 3]; // array

console.log(`Nome: ${nome}, Idade: ${idade}, Altura: ${altura}m, Estudante: ${estudante}`);

JavaScript é uma linguagem dinamicamente tipada, o que significa que você não precisa declarar explicitamente o tipo de uma variável. O tipo é determinado automaticamente quando o valor é atribuído.

Estruturas de Controle

// Estrutura condicional (if-else)
let idade = 18;

if (idade >= 18) {
    console.log("Você é maior de idade.");
} else {
    console.log("Você é menor de idade.");
}

// Operador ternário
let status = idade >= 18 ? "Maior de idade" : "Menor de idade";
console.log(status);

// Loop for
console.log("Contando de 1 a 5:");
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

// Loop for...of (para arrays)
const frutas = ["maçã", "banana", "laranja"];
for (const fruta of frutas) {
    console.log(fruta);
}

JavaScript oferece diversas estruturas de controle para tomada de decisão e iteração, incluindo condicionais como if-else e switch, e loops como for, while, do-while, além de for...of e for...in para iteração sobre arrays e objetos.

Funções

// Definindo funções
// Função tradicional
function saudacao(nome) {
    return `Olá, ${nome}! Bem-vindo ao JavaScript!`;
}

// Arrow function (ES6+)
const soma = (a, b) => a + b;

// Função com valor padrão
const multiplicar = (a, b = 1) => a * b;

console.log(saudacao("Carlos"));     // Olá, Carlos! Bem-vindo ao JavaScript!
console.log(soma(5, 3));             // 8
console.log(multiplicar(5));         // 5 (usa o valor padrão b=1)
console.log(multiplicar(5, 2));      // 10

// Funções de ordem superior
const numeros = [1, 2, 3, 4, 5];
const dobrados = numeros.map(num => num * 2);
console.log(dobrados);  // [2, 4, 6, 8, 10]

Em JavaScript, funções são objetos de primeira classe que podem ser atribuídos a variáveis, passados como argumentos e retornados de outras funções. Isso torna JavaScript uma linguagem poderosa para programação funcional.
