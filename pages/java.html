O que é Java?

Java é uma linguagem de programação orientada a objetos, multiplataforma e de alto nível, desenvolvida pela Sun Microsystems (agora pertencente à Oracle). Foi lançada oficialmente em 1995 e criada por James Gosling.

Java é conhecida pelo seu slogan "Write Once, Run Anywhere" (Escreva uma vez, execute em qualquer lugar), graças à sua capacidade de funcionar em diferentes plataformas sem recompilação. Isso é possível devido à Máquina Virtual Java (JVM).

Características principais do Java:

  • Orientação a objetos
  • Multiplataforma (independente de plataforma)
  • Fortemente tipada
  • Gerenciamento automático de memória (garbage collection)
  • Rico ecossistema de bibliotecas
  • Suporte a programação concorrente

Java é amplamente utilizada no desenvolvimento de aplicações empresariais, aplicativos Android, sistemas embarcados, aplicações web com frameworks como Spring, desenvolvimento de jogos, e muito mais. Sua estabilidade, segurança e comunidade ativa a tornam uma escolha excelente para diversos tipos de projetos.

Exemplos de Código Java

Olá Mundo

// Meu primeiro programa em Java
public class OlaMundo {
    public static void main(String[] args) {
        System.out.println("Olá, Mundo!");
    }
}

Este é o exemplo mais básico de um programa Java. O método System.out.println() exibe o texto "Olá, Mundo!" no console.

Variáveis e Tipos de Dados

// Variáveis em Java
public class Variaveis {
    public static void main(String[] args) {
        String nome = "Maria";    // string
        int idade = 25;          // inteiro
        double altura = 1.65;    // double
        boolean estudante = true; // boolean

        System.out.println("Nome: " + nome + ", Idade: " + idade + 
                          ", Altura: " + altura + "m, Estudante: " + estudante);
    }
}

Java é uma linguagem fortemente tipada, o que significa que você precisa declarar o tipo de cada variável. Isso ajuda a prevenir erros e melhora a performance.

Estruturas de Controle

// Estrutura condicional (if-else)
public class EstruturasControle {
    public static void main(String[] args) {
        int idade = 18;

        if (idade >= 18) {
            System.out.println("Você é maior de idade.");
        } else {
            System.out.println("Você é menor de idade.");
        }

        // Loop for
        System.out.println("Contando de 1 a 5:");
        for (int i = 1; i <= 5; i++) {
            System.out.println(i);
        }
    }
}

Java oferece estruturas de controle como if-else para tomada de decisão e loops para repetição de código, semelhantes a outras linguagens baseadas em C.

Métodos

// Definindo métodos
public class Metodos {
    public static void main(String[] args) {
        String mensagem = saudacao("Carlos");
        System.out.println(mensagem);

        System.out.println(soma(5, 3));  // 8
        System.out.println(soma(5));     // 5
    }

    static String saudacao(String nome) {
        return "Olá, " + nome + "! Bem-vindo ao Java!";
    }

    // Método com sobrecarga
    static int soma(int a, int b) {
        return a + b;
    }
    
    static int soma(int a) {
        return a;
    }
}

Em Java, métodos são blocos de código reutilizáveis que realizam tarefas específicas. Java suporta sobrecarga de métodos, que permite definir múltiplos métodos com o mesmo nome mas diferentes parâmetros.
