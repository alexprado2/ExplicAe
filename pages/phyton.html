O que é Python?

Python é uma linguagem de programação de alto nível, interpretada, de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991.

Python é conhecida por sua simplicidade e legibilidade, o que a torna uma excelente escolha para iniciantes em programação. Com sua sintaxe clara e direta, permite que os desenvolvedores se concentrem na resolução de problemas em vez de se preocupar com detalhes complexos da linguagem.

Características principais do Python:

  • Sintaxe clara e legível
  • Tipagem dinâmica
  • Interpretada
  • Orientada a objetos
  • Ampla biblioteca padrão
  • Comunidade ativa e grande ecossistema

Python tem ganhado popularidade em diversas áreas como ciência de dados, inteligência artificial, desenvolvimento web, automação, e muito mais. Sua versatilidade a torna uma ferramenta poderosa para praticamente qualquer tipo de projeto de software.

Exemplos de Código Python

Olá Mundo

# Meu primeiro programa em Python
print("Olá, Mundo!")

Este é o exemplo mais básico de um programa Python. A função print() exibe o texto "Olá, Mundo!" no console.

Variáveis e Tipos de Dados

# Variáveis em Python
nome = "Maria"  # string
idade = 25      # inteiro
altura = 1.65   # float
estudante = True  # boolean

print(f"Nome: {nome}, Idade: {idade}, Altura: {altura}m, Estudante: {estudante}")

Python suporta vários tipos de dados como strings, números e booleanos. Você não precisa declarar o tipo da variável explicitamente.

Estruturas de Controle

# Estrutura condicional (if-else)
idade = 18

if idade >= 18:
    print("Você é maior de idade.")
else:
    print("Você é menor de idade.")

# Loop for
print("Contando de 1 a 5:")
for i in range(1, 6):
    print(i)

Python oferece estruturas de controle como if-else para tomada de decisão e loops para repetição de código.

Funções

# Definindo uma função
def saudacao(nome):
    return f"Olá, {nome}! Bem-vindo ao Python!"

# Chamando a função
mensagem = saudacao("Carlos")
print(mensagem)

# Função com parâmetros padrão
def soma(a, b=0):
    return a + b

print(soma(5, 3))  # 8
print(soma(5))     # 5

Funções são blocos de código reutilizáveis que realizam tarefas específicas. Você pode definir funções com parâmetros e valores padrão.
